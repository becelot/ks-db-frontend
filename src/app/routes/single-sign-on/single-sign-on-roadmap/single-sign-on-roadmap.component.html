Single Sign-On is now in active development!

Roadmap:

  • Define data structures in backend
    • Must contain AWS account ids and display name for account
    • Must be groupable
    • A group may be associated to one or more users
    • A account can have multiple roles that the user can log into
  • Define and implement REST endpoints to retrieve the data
    • Mock data endpoint that initializes mock database with data
    • Endpoint that retrieves all groups and accounts associated with a user
    • Investigate how to actually sign in the user
  • Implement frontend Angular representation
    • Implement Account preview and option selector
    • Implement SSO flow to automatically sign in the user
